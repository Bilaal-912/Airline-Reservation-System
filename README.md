# Airline-Reservation-System
✈️ Airline Reservation System (Java GUI Project)
***📌 Overview***

The Airline Reservation System is a desktop-based Java application built using Java Swing that simulates a basic flight booking platform.
It allows users to securely log in, view available flights, select seats, and confirm bookings with real-time seat updates.

This project is designed as a lightweight academic prototype to demonstrate GUI development, event-driven programming, and Object-Oriented Programming (OOP) concepts in Java.

🚀*** Features***
 Secure Login System (Basic Authentication)
 ~View Available Flights (Flight No, Destination, Time, Seats)
 ~Select Flights and Book Seats
 ~Real-Time Seat Availability Updates
 ~User-Friendly GUI (Java Swing)
 ~Fast Performance (In-Memory Data Handling)
 
***Technologies Used***
i)Java
ii)Java Swing (GUI)
iii)OOP Concepts
iv)ArrayList (In-Memory Storage)

***How It Works***
User launches the application
Logs in using valid credentials
Views list of available flights
Selects a flight
Chooses number of seats
Confirms booking
Seat count updates instantly
